THE KLEIN-BOTTLE INTEGERS
q2 = 2   q3 = 3   product = 6   |F_6| = 13   budget = 19   exponent = 54   double = 108

These are the specific integers the framework selects, and each one is shown here
as a count or an area on the grid. They are assembled from the two base integers
q2 = 2 and q3 = 3 by the operations already drawn: laying out a length, filling a
rectangle, counting a Farey level, and stacking blocks. Every scorecard value in
layer V is later built out of these same integers, so this drawing collects them
in one place.

================================================================
THE TWO BASE LENGTHS  q2 = 2,  q3 = 3
================================================================

  q2 = 2 :  o---o---o          two unit steps
  q3 = 3 :  o---o---o---o       three unit steps

================================================================
PRODUCT  q2 x q3 = 6
================================================================

The product is the rectangle two units by three units, whose area is six.

  3 ·  ·          a 2-by-3 block
  2 ·  ·          area = 2 x 3 = 6
  1 ·  ·
    1  2

================================================================
FAREY COUNT  |F_6| = 13
================================================================

The Farey count is carried over from item 09, where level six holds thirteen
fractions. It is drawn here as thirteen marks.

  | | | | | | | | | | | | |     thirteen
  1 2 3 4 5 6 7 8 9 ...     13

================================================================
BUDGET  |F_6| + (q2 x q3) = 13 + 6 = 19
================================================================

The budget is the Farey count placed beside the product rectangle: thirteen marks
and six more, nineteen in all. This nineteen is the same denominator that the
cosmic budget tiles in item 13, where it splits as 1 + 5 + 13.

  13 (Farey)   +   6 (rectangle)   =   19

================================================================
EXPONENT  q2 x q3^3 = 2 x 27 = 54,  and DOUBLE = 108
================================================================

The exponent is q3 cubed, which is twenty-seven, taken twice, giving fifty-four.
The double exponent is two of those, one hundred eight. These appear in the scale
ladders of item 17, where R is 6 times 13 to the 54th and the cosmological
constant carries 13 to the minus 108th.

  q3^3 = 27   (a 3-by-3-by-3 block, drawn in item 11)
  exponent     = 2 x 27  = 54
  double       = 2 x 54  = 108

================================================================
ARRIVAL  -  the integers everything is built from
================================================================

The arrival is the set of integers itself: 2 and 3 at the base, their product 6,
the Farey count 13, the budget 19, the exponent 54, and the double 108. None of
these is drawn as motion, because each is a fixed count or area; the motion that
produced them is the laying-out and counting shown above. To redraw the set by
hand, mark the two base lengths, fill the six-rectangle, count the thirteen Farey
marks, set them side by side for nineteen, and stack twenty-seven twice for
fifty-four.
